# Job Portal System Backend

## Overview

A role-based Job Portal Backend developed using Java, Spring Boot, Spring Security, JWT Authentication, and MySQL. The system supports Admin, Recruiter, and Candidate roles with secure authentication and authorization.

## Features

* User Registration and Login
* JWT Authentication
* Role-Based Access Control (Admin, Recruiter, Candidate)
* Job Posting and Management
* Job Application Management
* Application Status Tracking
* Recruiter Dashboard APIs
* Candidate Profile Management
* RESTful API Architecture

## Technology Stack

* Java
* Spring Boot
* Spring Security
* Spring Data JPA
* JWT Authentication
* MySQL
* Maven
* Postman

## Modules

### Admin

* Manage Users
* Manage Recruiters
* Manage Jobs
* Manage Applications

### Recruiter

* Create Jobs
* Update Jobs

* View Applicants

### Candidate

* View Jobs
